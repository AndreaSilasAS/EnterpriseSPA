This is a Single Page Application (SPA) build with Angular 13, wich serves as fronted for a .NET 6 WebApi backend. The WebApi is responsible for managing companies, and the Angular frontend communicates with it through HTTP requests. Backend was developed with the Domain-Driven Design (DDD) pattern, it exposes various endpoints for retrieving, creating and updating companies.
